## Controle Proporcional Derivativo no controle do Braço Robótico.

Controle de posição e velocidade simultâneos
$$
\tau = K_p (q_{\text{des}} - q) + K_d (\dot{q}_{\text{des}} - \dot{q})
$$

Variáveis:
- T (Torque do Motor)
- qdes (Posição desejada)
- q (Posição Atual)
- 𝑞des ( Velocidade desejada)
- 𝑞 ( Velocidade Atual)
